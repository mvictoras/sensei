Supercomputing 2019

In 2019, we are presenting a tutorial on SENSEI on Sunday, November 17th. See the SC19 program for details. Some course materials are now available:

  • slides (PDF, 53 MiB, coming soon)
  • a virtual machine disk image (OVA, 7.3 GiB)

The disk image can be used with VirtualBox to try SENSEI out without having to build it yourself. However, once you decide to use it with your simulation, you should build it yourself to avoid the overhead of a virtual machine.

For those who attend the tutorial at Supercomputing, we plan to have temporary accounts and a small reservation set up on Cori @ NERSC, a real supercomputer, that you can use to try out SENSEI during the tutorial! We have found this a useful way to provide access to SENSEI without large downloads.
